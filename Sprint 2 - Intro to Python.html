Transforming variables with typecasting¶

The + operation can be used on numbers or strings. It will sum numbers or it will concatenate strings. An error will occur if you use the operation on a string with a number unless you change the type of variable. An example of a TypeError and successsful typecasting is shown below.

In [130]:
#this code produces a TypeError since a number cannot be concatenated with a string.
myDog + "is" + myDogAge + "years old."

In [131]:
#typecast a number to a string in order to concatenate.

'Logan is 8 years old.'
